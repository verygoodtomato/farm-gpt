# 기후변화 대응 농업 기술

## 1. 한국의 기후변화 현황
- 연평균 기온: 최근 30년간 1.4°C 상승
- 아열대 기후대 북상: 제주 → 남해안 → 중부 내륙
- 극한 기상: 폭염, 집중호우, 한파 빈도 증가
- 재배적지 변화: 사과(대구→영주→강원), 배(나주→충남)

## 2. 이상기후 유형별 대응

### 폭염 (35°C 이상)
- **시설원예**: 차광막 가동, 포그 냉방, 환기 강화
- **노지**: 짚멀칭, 관수량 증가, 해가림 설치
- **과수**: 미세살수, 반사필름, 과실 봉지 씌우기
- **수확기 대응**: 새벽/저녁 수확, 예냉 처리

### 집중호우/태풍
- **사전**: 배수로 정비, 하우스 보강, 방풍망
- **사후**: 도복 작물 세움, 엽면시비, 살균제 방제
- **토양 관리**: 객토, 유기물 투입, 배수 개선

### 이상 저온/늦서리
- **보온대책**: 보온커튼, 소형터널, 부직포 피복
- **가온**: 열풍기, 연소식 난방
- **작물 선택**: 내한성 품종 선택

### 가뭄
- **관수 효율화**: 점적관수, 지중관수
- **토양 수분 보전**: 멀칭, 부초
- **내건성 품종**: 건조 저항성 유전자원 활용

## 3. 이상기후 적응형 재배 기술

### 작형 조절
- 정식/파종 시기 조정 (온도 변화 반영)
- 작기 이동: 촉성재배 → 반촉성 → 억제재배
- 이모작/다모작 확대

### 품종 개량
- 내서성/내한성 품종 개발
- 내병성 품종 보급
- 단일/장일 무감응성 품종

### 시설 개선
- 내재해형 온실 구조 (풍속 50m/s 대응)
- 이중피복: 에너지 절감 30-50%
- 스마트 환기 시스템

## 4. 기후 데이터 활용

### 기상청 API 활용
- 동네예보: 3시간 단위 기온, 강수, 풍속
- 중기예보: 10일간 전망
- 농업기상관측: 농업 특화 기상 정보

### 농업기상 지수
- **생육도일(GDD)**: 작물 생육 단계 예측
  - GDD = Σ((일최고기온+일최저기온)/2 - 기저온도)
- **한랭피해지수**: 동해 위험도 평가
- **병해충 발생 예찰 모델**: 온습도 기반 발생 확률

## 5. 탄소중립 농업
- 저탄소 농법: 무경운, 피복작물, 바이오차
- 에너지 절감: 히트펌프, 지열, 태양광
- 스마트 시비: 정밀 양분 관리로 N2O 저감
- 탄소 흡수: 농경지 토양 유기탄소 증대
