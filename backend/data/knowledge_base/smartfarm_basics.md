# 스마트팜 기초 가이드

## 1. 스마트팜이란
ICT(정보통신기술)를 온실, 축사, 과수원 등에 접목하여 원격·자동으로 작물의 최적 생육환경을 유지·관리하는 농업 시스템.

## 2. 스마트팜 핵심 구성요소

### 센서 시스템
- **환경 센서**: 온도, 습도, CO2, 광량, 풍향/풍속
- **토양 센서**: 토양온도, 수분, EC, pH
- **생체 센서**: 줄기직경, 엽온, 과실중량
- **기상대**: 외부 기온, 강수량, 일사량

### 제어 장치
- **천창/측창**: 환기 제어
- **보온커튼**: 야간 보온
- **차광막**: 하절기 차광
- **CO2 발생기**: 탄산시비
- **난방기**: 온풍난방, 온수난방
- **냉방기**: 패드앤팬, 포그
- **양액기**: 급액 자동 제어
- **보광등**: LED, 고압나트륨등
- **순환팬**: 공기 순환

### 통합 제어 시스템
- 센서 데이터 수집 → 분석 → 제어 명령
- 복합환경제어: 온도+습도+CO2+광량 연동
- 자율제어 vs 규칙기반제어 vs AI 제어

## 3. 환경 제어 원리

### 온도 제어
- **난방**: 온풍난방(즉각 반응), 온수난방(균일 분포)
- **냉방**: 환기(자연/강제), 차광, 포그 시스템
- **변온 관리**: 주간/야간 온도차 활용 (DIF)
  - Positive DIF: 주간 > 야간 → 줄기 신장 촉진
  - Negative DIF: 주간 < 야간 → 줄기 신장 억제
  - Zero DIF: 주간 = 야간 → 균형 생장

### 습도 제어
- **가습**: 포그, 미스트 시스템
- **제습**: 환기, 난방, 제습기
- **포차(VPD) 관리**: 작물 증산 속도 조절
  - VPD = 포화수증기압 - 실제수증기압
  - 적정 VPD: 0.5-1.2 kPa (작물별 차이)
  - VPD 높음 → 증산 과다 → 시들음
  - VPD 낮음 → 증산 부족 → 병해 위험

### CO2 관리
- 대기 중 농도: 약 420ppm
- 작물 광합성 최적: 700-1,200ppm
- 공급 방법: 순수 CO2, 연소식(등유), 유기물 발효
- 공급 타이밍: 일출 후 환기 전까지

### 광 관리
- 자연광 활용 극대화: 피복재 투광률 관리
- 보광: LED (적색+청색), 고압나트륨등
- 차광: 하절기 고온 대응, 가동차광막
- DLI(일적산광량) 관리: 작물별 목표 설정
  - 딸기: 12-17 mol/m²/day
  - 토마토: 20-30 mol/m²/day

## 4. 데이터 기반 의사결정

### 수집 데이터
- 환경 데이터: 5분 간격 센서 데이터
- 생육 데이터: 주 1-2회 수동 측정
- 생산 데이터: 수확량, 품질 등급
- 에너지 데이터: 난방유, 전력 사용량

### AI 활용 영역
- **환경 예측**: 온실 내부 환경 예측 모델
- **생육 예측**: 수확 시기, 수량 예측
- **최적 제어**: 강화학습 기반 환경 제어
- **병해충 진단**: 이미지 인식 기반 자동 진단
- **에너지 최적화**: 난방/냉방 에너지 절감

## 5. 한국형 스마트팜 세대 구분

### 1세대
- 시설 원격 모니터링·제어
- 스마트폰으로 환경 확인 및 제어
- 단순 임계값 기반 제어

### 2세대
- 데이터 기반 정밀 생육 관리
- 환경-생육 데이터 연동 분석
- AI 기반 최적 환경 제어

### 3세대
- 로봇, 자율주행 등 무인 자동화
- 에너지 자급, 자원 순환
- 완전 자율 제어 시스템

## 6. 에너지 효율 관리
- 보온커튼 적시 개폐: 에너지 30-50% 절감
- 축열조 활용: 주간 잉여열 저장 → 야간 활용
- 히트펌프: 난방비 50-60% 절감
- 지열 활용: 연중 안정적 열원
- 태양광: 온실 지붕 일부 활용 (투광률 저하 주의)
