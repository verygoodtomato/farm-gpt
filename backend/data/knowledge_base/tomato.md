# 토마토 재배 가이드

## 1. 품종 선택
### 대과종
- **도태랑**: 국내 대표 품종. 과중 200-250g. 당도 5-6 Brix.
- **레드앤그린**: 경도 우수. 겨울 재배 적합.
- **TY계열**: 황화잎말림바이러스 저항성 품종.

### 중과종 (방울토마토)
- **꼬꼬**: 과중 15-20g. 당도 8-10 Brix.
- **미니찰**: 높은 당도(10-12 Brix). 식감 우수.
- **대추형**: 스테비아 토마토로 인기. 당도 12 Brix 이상.

## 2. 재배 환경
### 온도 관리
- 발아 적온: 25-30°C
- 육묘기: 주간 25-28°C, 야간 15-18°C
- 생육 적온: 주간 22-28°C, 야간 13-18°C
- 착과 적온: 주간 25°C, 야간 15°C
- 착색 적온: 20-25°C (30°C 이상 시 착색 불량)

### 습도 관리
- 적정 습도: 60-70%
- 과습 시 잎곰팡이병, 역병 발생 위험
- 건조 시 꽃떨이 현상, 배꼽썩음과 발생

### 광 관리
- 최소 일조시간: 6-8시간
- 광포화점: 70,000 lux
- 겨울철 보광등 사용 권장

### CO2 관리
- 적정 농도: 800-1,200ppm
- 광합성 효율 극대화
- 밀폐 후 공급, 환기 시 중단

### 양액 관리
- pH: 5.5-6.5
- EC: 2.0-3.5 dS/m
- 급액 횟수: 일사량 기반 자동 관리

## 3. 주요 병해충
### 병해
- **잎곰팡이병**: 잎 뒷면 올리브색 곰팡이. 환기 중요. 저항성 품종 사용.
- **역병**: 갑작스런 시들음. 배수 관리, 토양 소독.
- **황화잎말림바이러스(TYLCV)**: 담배가루이 매개. 저항성(TY) 품종 필수.
- **풋마름병**: 세균성. 뿌리 통한 감염. 접목묘 사용.
- **흰가루병**: 건조할 때 발생. 적정 습도 유지.

### 해충
- **담배가루이**: TYLCV 매개. 황색끈끈이트랩, 천적 방사.
- **아메리카잎굴파리**: 잎에 굴을 만듦. 기생봉 천적 활용.
- **총채벌레**: 과실 표면 손상. 끈끈이트랩, 천적.
- **온실가루이**: 잎 뒷면 서식. 천적(온실가루이좀벌) 활용.

## 4. 수확 및 품질 관리
- 대과종: 완숙 수확 (80% 이상 착색)
- 방울토마토: 100% 착색 후 수확
- 저장 온도: 10-13°C (저온장해 주의, 5°C 이하 금지)
- 저장 습도: 85-90%

## 5. 스마트팜 제어 포인트
- 일출 후 1시간 내 환기 시작 (결로 제거)
- 광합성 최적화: CO2 + 온도 + 광량 연동 제어
- 야간 변온 관리: 전반야 15°C → 후반야 10°C (호흡량 감소, 당도 향상)
- 착과 불량 시 진동수분 또는 호르몬 처리
