# 딸기 재배 가이드

## 1. 품종 선택
### 주요 품종
- **설향**: 국내 재배면적 80% 이상 차지. 과형이 원추형이고 당도가 높음 (11-13 Brix). 내한성 강하고 수량성 우수.
- **매향**: 수출용 품종. 경도가 높아 유통성 우수. 당도 10-12 Brix.
- **죽향**: 대과형 품종. 과중 20-30g. 향이 좋고 당도 높음.
- **킹스베리**: 대과형 고당도 품종. 과중 30-40g.
- **비타베리**: 비타민C 함량이 높은 기능성 품종.

## 2. 재배 환경
### 온도 관리
- 정식 후 활착기: 주간 25-28°C, 야간 12-15°C
- 생육기: 주간 23-25°C, 야간 8-10°C
- 개화기: 주간 20-25°C, 야간 5-8°C
- 과실 비대기: 주간 20-23°C, 야간 5-7°C
- 착색기: 주간 18-22°C, 야간 3-5°C (주야간 온도차 15°C 이상 유지)

### 습도 관리
- 적정 습도: 60-70%
- 80% 이상 시 잿빛곰팡이병 발생 위험 증가
- 환기로 습도 조절, 특히 개화기에 중요

### 광 관리
- 일조시간: 12-14시간 권장
- 보광등 사용 시 6,000-10,000 lux
- 11월-2월 부족한 일조량 보충 필요

### CO2 관리
- 적정 농도: 700-1,000ppm
- 광합성 촉진을 위해 일출 후 2-3시간 집중 공급
- 환기 시 외부 CO2 농도(400ppm)로 감소하므로 타이밍 조절 필요

### 배지 및 양액
- pH: 5.5-6.5
- EC: 0.8-1.2 dS/m (생육기), 1.0-1.5 dS/m (과실 비대기)
- 급액량: 생육 단계별 차등 관리

## 3. 주요 병해충
### 병해
- **잿빛곰팡이병**: 고온다습 시 발생. 회색 곰팡이가 과실, 꽃에 발생. 환기 중요.
- **흰가루병**: 잎 표면에 흰 가루 형태. 온도 15-25°C, 습도 높을 때. 저항성 품종 선택.
- **탄저병**: 고온다습기(7-9월) 육묘기에 주로 발생. 크라운부 갈변. 건전묘 사용.
- **시들음병**: 관부가 갈변하고 포기째 시듦. 토양 소독, 저항성 품종 사용.
- **역병**: 뿌리와 크라운부 갈변. 배수 개선, 과습 방지.

### 해충
- **점박이응애**: 잎 뒷면 가해. 천적(칠레이리응애) 방사 효과적.
- **총채벌레**: 꽃과 과실 가해. 끈끈이트랩 설치, 천적 활용.
- **진딧물**: 신초 가해, 바이러스 매개. 천적(진디벌) 활용.
- **작은뿌리파리**: 뿌리 가해. 황색끈끈이트랩으로 예찰.

## 4. 수확 및 품질 관리
- 수확 적기: 과실 표면의 80-90% 착색 시
- 수확 시간: 아침 기온이 낮을 때 (과실 경도 유지)
- 저장 온도: 1-3°C
- 저장 습도: 90-95%
- 유통 기간: 냉장 보관 시 5-7일
